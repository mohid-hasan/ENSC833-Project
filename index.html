ENSC 833 Project

Performance Evaluation of Machine Learning Models Using IODA Data

Name: Mohid Hasan

Abstract

Internet anomalies may result from malware outbreaks, infrastructure failures, and conflict-driven connectivity disruptions. Early detection is essential for maintaining network stability, improving cybersecurity visibility, and ensuring resilience during crises. This project evaluates eight machine learning models for anomaly detection using Internet Outage Detection and Analysis (IODA) datasets within a sliding-window framework that captures temporal dependencies.

Two case studies, Iran and Cuba, are analyzed to represent both severe shutdowns and subtle connectivity degradations. The models are evaluated using ROC-AUC, PR-AUC, Accuracy, F1-score, confusion-matrix-based metrics, Precision, Sensitivity, and training time. Results show that temporal models such as RNN-GRU and RNN-Bi-LSTM provide strong and balanced detection performance, while Broad Learning System variants achieve near-top accuracy with significantly lower training cost, making them suitable for real-time deployment. The results also highlight that outage severity affects detection difficulty and that sliding-window modeling effectively captures temporal disruption patterns.
