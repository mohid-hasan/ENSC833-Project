ENSC 833 Project

Machine-Learning Analysis of Recent Internet Outages Using IODA Data

Name: Mohid Hasan

Abstract

The Internet Outage Detection and Analysis (IODA) platform provides multi-signal time-series data to monitor large-scale Internet disruptions. Prior work has shown that sliding-window machine-learning methods can detect outage events in selected case studies, but generalization to new outage scenarios remains an open question.

This project analyzes recent IODA outage events using raw time-series data and evaluates multiple machine-learning models under a consistent sliding-window framework. The goal is to assess robustness across different outage types and identify practical approaches for reliable anomaly detection.
